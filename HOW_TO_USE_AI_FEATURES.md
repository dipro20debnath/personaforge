# 🚀 How to Use AI Features - Quick Start

## ✅ AI Features Now Active on Dashboard!

All AI features are now integrated and working. Here's how to use each one:

---

## 1. **🤖 AI Personal Coach** (Floating Widget)

### How to Access:
- Look for the **purple floating button** in the bottom-right corner of your screen
- Click it to open the chat widget
- It appears on **every page**

### How to Use:
1. Click the purple circle icon (bottom-right)
2. The coach widget opens with a greeting
3. Type your question or request:
   - "How do I stay motivated?" 
   - "What should I work on today?"
   - "Help me with my goals"
4. Get personalized AI-powered responses
5. Click minimize (−) to collapse, or X to close

### Tips:
- ✨ Auto-generates greeting based on your stats
- 💬 Available 24/7 for support
- 🎯 Context-aware advice
- 📱 Mobile-friendly

---

## 2. **✨ Smart Insights** (Right Sidebar on Dashboard)

### How to Access:
- Go to **Dashboard** page
- Look at the **right sidebar**
- You'll see colorful insight cards

### How to Use:
1. Review the insights cards
2. Each card shows:
   - 🎉 Achievements ("7-Day Streak!")
   - 💡 Opportunities ("Skill Gap")
   - ⚠️ Warnings ("Low Activity")
   - 🎯 Milestones ("67% to Goal")
3. Click the action button (e.g., "Add Habit")
4. Click ✕ to dismiss insights

### Examples:
- **Achievement**: "🎉 7-Day Streak! You're on fire!"
- **Opportunity**: "💡 Add Leadership Skills - mentioned in goals"
- **Warning**: "⚠️ Activity down 30% this week"
- **Milestone**: "🎯 67% to next goal - keep going!"

## 3. **💡 Quick Suggestions** (Dashboard Main Content)

## ❓ Troubleshooting

### Buttons Not Working?

**Problem**: Buttons don't respond when clicked
**Solution**:
1. Refresh the page (Ctrl+R or Cmd+R)
2. Check browser console (F12) for errors
3. Make sure you're logged in
4. Try in incognito/private mode

### Components Not Showing?

**Problem**: I don't see AI features
**Solution**:
1. Clear browser cache
2. Hard refresh (Ctrl+Shift+R)
3. Try different browser
4. Check if logged in

### Chat Widget Not Opening?

**Problem**: Purple button doesn't open chat
**Solution**:
1. Make sure button is visible (bottom-right)
2. Click directly on the button
3. Try refreshing page
4. Check browser console for JS errors

### API Errors?

**Problem**: Seeing "Failed to fetch" errors
**Solution**:
1. Check your internet connection
2. Wait a few seconds and try again
3. Make sure backend is running
4. Check firebase token is valid
