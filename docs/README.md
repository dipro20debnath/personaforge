# 🔥 PersonaForge — Personality Management Platform

## Quick Start
```bash
node setup.mjs
cd personaforge
npm install
npm run install:all
npm run dev
```

- **Frontend**: http://localhost:3000 (Next.js 14 + TypeScript + Tailwind)
- **Backend**: http://localhost:5000 (Express + SQLite)

## Features (MVP)
- 🔐 Auth (JWT + bcrypt)
- 👤 Profile Management
- 🧠 Big Five Personality Assessment (IPIP)
- 📊 Skill Tracking + World Benchmark
- 🎯 SMART Goal Setting
- ✅ Habit Tracker with Streaks
- 📝 Daily Journal with Tags & XP
- 📚 Learning Path Recommendations
- 🗺️ Career Map
- 📄 CV Builder + PDF Export
- 💚 Mental Health Resources
- 🛡️ Privacy Center (Export/Delete Data)
- 🌙 Dark/Light Mode
- 📱 Fully Responsive
- 🔔 Smart Notifications
- 📈 Dashboard Overview
